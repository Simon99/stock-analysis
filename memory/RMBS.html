Rambus RMBS · 美

DRAM配套IC(RCD/PMIC/SPD)+記憶體IP · 中游·DRAM介面IC

「IP授權占比高,吃不到漲價財,漲勢明顯落後同業」

Rambus Q1 2026營收1.802億美元(YoY+8%),產品營收(RCD/PMIC等配套IC)年增15%至8,800萬美元,表現優於整體公司平均,但仍落後記憶體介面晶片市場約29.5%的產業CAGR,顯示正被Montage Technology、Renesas等對手分食DDR5升級周期紅利。更關鍵的是,公司近4成營收(royalty約6,960萬美元)來自固定費率的IP授權金,不論DRAM現貨價漲跌都大致固定,加上配套IC本質是「隨模組數量出貨」而非「隨記憶體均價調整」的商業模式,使Rambus在2026年記憶體全面性缺貨、美光/SK海力士/三星DRAM現貨價齊漲的行情中,幾乎完全吃不到漲價財。股價從2月高點174.10美元回落至7月中的102.81美元(跌幅逾4成),RS6M明顯落後三星、美光等同業,估值面trailing PE高達53-56倍、前瞻PE 37.98倍(較半導體業中位數高出約10%),股價淨值比逾6-7倍,顯示市場仍給予其AI資料中心配套IC與CXL/HBM4E IP授權成長性溢價,但相對弱勢已反映「單位量驅動、非價格驅動」的結構性弱點,存貨QoQ+32%、週轉天數拉長至146天則是為因應2027年前供應鏈缺貨而預先囤貨,短線需留意去化速度。

1 財務健康留意

營收YoY+8%溫和/產品營收YoY+15%較強/毛利率79.8%
→ Q1 2026總營收1.802億美元(YoY+8%),其中產品(配套IC)營收8,800萬美元(YoY+15%)、royalty授權金6,960萬美元、合約及其他收入2,260萬美元;毛利率高達79.8%、現金部位7.861億美元皆屬健康,惟營收成長速度遠不及記憶體晶片產業整體29.5%的CAGR。

trailing PE≈53-56x/前瞻PE≈38x(業界中位數34.5x)/PBR≈6.5-7x
→ GuruFocus估算trailing PE 56.04倍、較同業中位數高49%,前瞻PE 37.98倍亦較半導體業中位數34.50倍高出約10%,PBR約6.5-7倍;GuruFocus GF Value模型與DCF分析皆判定股價「適度至明顯高估」(約高估30%左右)。

股價RS6M≈-5%明顯落後同業/營收YoY+8%溫和/產品營收YoY+15%
→ 股價從2月174.10美元高點回落至7月中約103美元(跌幅逾40%),在記憶體股全面上漲的2026年明顯跑輸大盤與美光、SK海力士等同業;儘管配套IC產品線成長動能相對較強(YoY+15%),但公司整體營收成長仍屬溫和(YoY+8%),不足以支撐股價追上同業漲勢。

RS6M≈-5%落後/7/13單日重挫8%/分析師仍偏多(11 buy、3 hold)
→ 近期股價呈現高檔震盪重挫,7月13日單日下跌8%至103.02美元,4月Q1財報公布後亦因獲利成長不如預期單日重挫;惟華爾街分析師評等仍偏多(11家買進、3家持有),Rosenblatt於近期將目標價由150美元上調至165美元、維持買進評等,目標價共識約135-145美元。

5 風險旗標貴/風險

商業模式偏IP授權/單位量驅動非價格驅動/不吃記憶體漲價財
→ Rambus近4成營收來自royalty固定授權金,配套IC產品營收亦主要隨模組出貨「單位數量」而非DRAM現貨「價格」變動,使其在本輪記憶體缺貨漲價循環中無法像上游DRAM原廠(美光/SK海力士/三星)直接享受ASP飆升紅利;此外新增DRAM產能多優先投入HBM,反而排擠RDIMM/MRDIMM等Rambus配套IC對應的模組供給,構成營收成長的結構性限制。存貨QoQ+32%、週轉天數拉長至146天為因應2027年前供應鏈持續緊張而預先囤貨,屬策略性佈局但仍須留意去化風險。

6 技術/循環位置留意

股價距52週高點174.10美元已回落約41%,仍在低點62.81美元之上
→ 股價自2月52週高點174.10美元回落至7月中約103美元附近(距高點約-41%),雖仍遠高於52週低點62.81美元,惟技術面呈現高檔修正整理格局,short-term動能疲弱,尚未見到明確落底訊號。

產業鏈傳導

Rambus橫跨兩塊業務:一是DDR5配套IC硬體產品線,涵蓋第六代RCD(RCD06,暫存器時脈驅動器)、DDR5 Server PMIC(電源管理IC)、SPD溫感等,DDR5 RCD市佔約40%、PMIC目標拉升至約20%,並新推出SOCAMM2晶片組搶進以LPDDR5X為基礎的AI伺服器記憶體模組市場;二是IP授權業務,包括CXL 3.1控制器IP、記憶體PHY IP、以及新發表的HBM4E控制器IP,鎖定AI資料中心對高頻寬記憶體的需求,以royalty固定授權金模式收取現金流。核心邏輯是隨DDR4→DDR5、RDIMM→MRDIMM的世代轉換提高「每模組內容價值」(content per module)與市佔率,而非直接受惠DRAM現貨價格上漲,這也是為何在本輪記憶體全面性缺貨漲價循環中,Rambus股價表現明顯落後美光、SK海力士等上游DRAM原廠的核心原因。

🖥 伺服器 / 資料中心 design-in

資料中心/AI伺服器DDR5配套IC設計導入現況:第六代RCD(RCD06)搭配DDR5 Server PMIC組成完整晶片組,已切入主要記憶體模組廠商的RDIMM/MRDIMM設計;7月初發布DDR5 9600 Server RDIMM晶片組進一步鎖定下世代AI與資料中心平台的頻寬需求;新推出的SOCAMM2晶片組支援JEDEC標準LPDDR5X(最高9.6 Gb/s),鎖定低功耗、高頻寬、體積精簡的AI伺服器記憶體模組應用,為公司規劃中LPDDR系列伺服器模組晶片組家族的首發產品。CXL 3.1控制器IP與HBM4E控制器IP則主打AI資料中心對高頻寬互連與記憶體擴充的需求,惟這類IP授權多處於設計導入(design-in)早期階段,尚未大規模挹注營收,是公司未來成長性題材而非現階段主要獲利引擎。

財報 / 循環位置

Q1 2026(截至3月)GAAP總營收1.802億美元,較去年同期1.667億美元成長約8%;營收結構為產品(配套IC)8,800萬美元(占比48.8%,YoY+15%)、royalty授權金6,960萬美元(占比38.7%)、合約及其他收入2,260萬美元(占比12.5%)。毛利率高達79.8%,單季營運現金流8,320萬美元,現金及有價證券合計7.861億美元,財務體質穩健。存貨因應2027年前預期持續緊張的供應鏈而策略性增加(QoQ約+32%,單季增加約1,400萬美元),使週轉天數拉長至約146天。Q2 2026財測:licensing billings 7,600-8,200萬美元、royalty營收7,200-7,800萬美元、產品營收9,500-1.01億美元、Non-GAAP EPS 0.65-0.73美元,總營收指引約1.92-1.98億美元(QoQ約+7-10%)。公司业务定位橫跨「硬體配套IC」(隨DDR5/MRDIMM世代轉換與市占率成長)與「IP授權」(隨CXL/HBM4E等新規格逐步放量,但授權金相對固定、不隨DRAM現貨價連動)兩塊,是其股價在本輪記憶體漲價循環中明顯跑輸美光、SK海力士等上游DRAM原廠的根本原因;下一次財報(Q2 2026)將於7月27日公布,為觀察配套IC市占率與IP授權進展的下一個關鍵時點。
