怎麼讀 · 紀律

• 先分 DRAM vs NAND——兩個不同市場:TrendForce Server DRAM 3Q26 +13-18% vs SLC NAND 2H26 +120-170%,量級差很多。
• 庫存消化是核心次級指標:存貨↑囤貨+營收↑=需求強(健康);存貨↓消化+營收↑=最健康(賣爆);存貨↑但營收轉弱=循環反轉警訊。
• 原廠是循環驅動、台廠是下游:看原廠(三星/SK/美光)定調 DRAM/NAND 報價與擴產,台廠(群聯/南亞科…)跟隨。
• 裂縫要盯:潘健成自曝現貨(LPDDR4)已回跌五成 → 現貨先鬆是傳統領先訊號;原廠合約報價何時鬆是下一個。
